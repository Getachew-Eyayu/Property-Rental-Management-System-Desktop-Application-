🏠 Property Rental Management System

A desktop-based Property Rental Management System built using Java (OOP), JavaFX (FXML), and MySQL. The system is designed to reduce paper-based rental management by digitizing property, tenant, and payment records.

📌 Overview

Managing rental properties manually can lead to lost documents, slow processing, and errors. This system provides a digital solution to manage properties, tenants, rentals, and payments efficiently in one centralized application.

🚀 Features

🏢 Add, update, delete, and view properties 👤 Tenant registration and management 📄 Rental agreement management 💰 Payment tracking and records 🔍 Search and filter system for fast access 📊 Organized data display using tables 🔐 Structured and secure database handling 🛠 Technologies Used Java (OOP) JavaFX (FXML) MySQL JDBC (Database connectivity) 🎯 Purpose of the Project

The main goal of this project is to:

Reduce paperwork in rental management Improve data organization and accessibility Automate manual rental operations Provide a user-friendly desktop solution for property management 🧩 System Modules Property Management Module Tenant Management Module Rental Agreement Module Payment Management Module Search & Reporting Module 💡 Problem Solved

Traditional rental systems rely heavily on paper documents, which causes:

Data loss Difficulty in tracking payments Inefficient communication Time-consuming management

This system solves these issues by digitizing and centralizing all rental operations.
